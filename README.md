# Spring Kit

Clone do Spring Initializr com suporte para versões LTS do Java (8, 11, 17, 21, 25) e versões compatíveis do Spring Boot.

## Características

- ✅ Suporte para Java LTS: 8, 11, 17, 21, 25
- ✅ Versões compatíveis do Spring Boot para cada versão do Java
- ✅ Interface moderna com Thymeleaf
- ✅ Geração de projetos Maven e Gradle
- ✅ Dependências mais utilizadas do Spring
- ✅ Download direto do projeto em formato ZIP

## Tecnologias

- Spring Boot 3.2.0
- Thymeleaf
- Lombok
- Maven

## Como executar

```bash
mvn spring-boot:run
```

Acesse: http://localhost:8080

## Versões compatíveis

### Java 8 (LTS)
- Spring Boot: 2.0.9, 2.1.18, 2.2.13, 2.3.12

### Java 11 (LTS)
- Spring Boot: 2.3.12, 2.4.13, 2.5.15, 2.6.15, 2.7.18

### Java 17 (LTS)
- Spring Boot: 2.7.18, 3.0.13, 3.1.12, 3.2.5, 3.3.0

### Java 21 (LTS)
- Spring Boot: 3.2.5, 3.3.0

### Java 25 (LTS - Future)
- Spring Boot: 3.4.0

## Dependências disponíveis

- Spring Web
- Thymeleaf
- Spring Data JPA

- H2 Database
- MySQL Driver
- PostgreSQL Driver
- Spring Security
- Spring Boot DevTools
